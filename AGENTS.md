# Consultant AI

A task-oriented, agentic platform that abstracts complex workflows into a Kanban-first experience. Agents discover, invent, and execute skills via a standardized protocol.

**Environment:** PowerShell terminal. Dev server is always running. TanStack app lives in `/app`.

**UI:** Use shadcn/ui components. Kanban board: @dnd-kit (core, sortable, utilities).

**Guidelines:** 
- Always ask the user for clarification if something is unclear.
- ALWAYS run a linting check (e.g., `npm run lint`) after making significant or structural code changes to catch type/import errors early.
